TempleOS — the real 64-bit OS, booting in your browser

This boots the complete, unmodified TempleOS V5.03 on a 64-bit QEMU whose CPU core is compiled to WebAssembly — JIT-translating the guest's x86-64 to WASM. Real kernel, real HolyC JIT, graphics, sound, keyboard & mouse. source & build notes →

Notes: needs a desktop browser with SharedArrayBuffer (cross-origin isolation — provided here by the bundled service worker, or by COOP/COEP headers on hosts that support them). The default boots the installed hard disk, restored instantly from a snapshot (~4s) — no compile wait, no prompts. Click a window, then type (keyboard & mouse work; capital letters too); TempleOS routes keys to the focused window. ↻ Restart reloads the clean desktop in a few seconds — use it any time TempleOS gets into a weird state (e.g. a mistyped command pops up its built-in debugger; that’s authentic TempleOS, and Restart clears it). 🧭 Take the Tour opens the built-in Welcome guide. Note: TempleOS’s native interactive pop-up tour doesn’t get input focus under the software renderer, so the tour button opens the same intro documentation instead (navigable, with links).
